# 📟 GeekQuiz-JS v1.0
> **Turn your technical articles into interactive terminal challenges.**

GeekQuiz-JS is a lightweight, zero-dependency JavaScript engine that automatically generates "fill-in-the-blank" quizzes by analyzing your blog post's content. It’s designed specifically for tech bloggers who want to increase dwell time and engage their audience with a cyberpunk aesthetic.

## 🚀 Key Features
- **Auto-Generation:** No database or manual input. It scans your `<p>` tags and creates challenges on the fly.
- **Cyberpunk UI:** A sleek, terminal-inspired interface that fits the "hacker" aesthetic.
- **Dynamic Ranking:** Assigns ranks (Recruit, SysAdmin, Architect) based on user performance.
- **Social Propagation:** Built-in sharing logic for X, LinkedIn, Telegram, and more.
- **Lightweight:** Pure Vanilla JS. No jQuery, no bloat.
