# 📟 GeekQuiz-JS
> **Autonomous Cyberpunk Quiz Engine for Technical Blogs.**

GeekQuiz-JS è un motore di quiz leggero che analizza automaticamente i tuoi articoli e genera sfide interattive per aumentare il tempo di permanenza sul sito.

## 🛠 Come installarlo
1. Carica `geekquiz.js` sul tuo server.
2. Aggiungi `<div id="cg-auto-quiz"></div>` dove vuoi che appaia il quiz.
3. Includi lo script: `<script src="geekquiz.js"></script>`.
